📌 What is a Computer Program?

🔍 Definition A computer program is a set of logical instructions written in a programming language that directs a computer to perform specific tasks. Without programs, computers are useless.
🧠 Analogy: Recipe Just like a recipe gives step‑by‑step cooking instructions, a program tells the CPU exactly what to do (input, processing, output).
⚡ Key fact Programs can be written in different languages (Python, C++, Java) but achieve the same result if logic is correct.

📜 Core Components of a Program

  • Algorithms: logical step‑by‑step procedure.
  • Data Structures: organizing data efficiently.
  • Syntax: grammatical rules.
  • Compilation/Interpretation: translation to machine code.
# Example: Simple Python program
name = input("Enter your name: ")
print("Hello,", name, "! Welcome to programming.")
✅ Benefits: Automation, precision, solving complex problems.
⚠️ Consequence of Bad Program: Crashes, security vulnerabilities, wrong calculations.
